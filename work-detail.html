Tweet Emotion Detection using LLMs

NLP Emotion Detection Machine Learning

Classified tweet data into multiple emotions using various Large Language Models (LLMs) and explore different parameter efficient fine-tuning (PEFT) methods.

  • Implemented and compared multiple LLMs including DistilBERT, RoBERTa, and Flan-T5.
  • Explored advanced PEFT methods such as LoRA, IA3, and QLoRA on billion-parameter models.
  • Achieved a top Macro F1-score of 0.6312 using DistilBERT.
  • Successfully applied PEFT techniques to achieve comparable performance (F1-score: 0.59833) with significantly reduced computational resources.

This project showcases the ability to work with cutting-edge NLP technologies and optimize them for real-world applications. The insights gained contribute to more efficient and accurate sentiment analysis tools, with potential applications in social media monitoring, customer feedback analysis, and public opinion tracking.

Technologies: Deep Learning & NLP, PEFT Techniques, Data Analysis, Performance Optimization, ML Ops

Prediction of Risk of Heart Disease

Machine Learning Classification

Developed a sophisticated machine learning system to predict the risk of heart disease using data from the 2015 Behavioral Risk Factor Surveillance System (BRFSS) survey. This project addressed a critical healthcare challenge, as heart disease is a leading cause of death in the United States.

  • Implemented and compared multiple machine learning models including Logistic Regression, Decision Trees, Support Vector Machines (SVM), and K-Nearest Neighbors (KNN).
  • Addressed significant class imbalance using advanced techniques like Synthetic Minority Over-Sampling Technique (SMOTE).
  • Identified key predictors of heart disease risk, including age, general health, high blood pressure, and high cholesterol.

This project contributes to early detection and prevention of heart disease by providing a tool for risk assessment. The insights gained can help healthcare professionals identify high-risk individuals and implement targeted interventions, potentially reducing the burden of heart disease on individuals and the healthcare system.

Technologies: Data Preprocessing, Machine Learning, Data Analysis, Model Evaluation, Scikit-Learn, Matplotlib, Pandas, Numpy

Corporate Travel Management System

MySQL RDBMS Database Design

Designed a comprehensive database management system for corporate travel to provide a one-stop platform for automating the travel management of companies. This model covers two major workflows of business travel for corporate employees:

  • Assigning and processing raised travel requests to the concerned managers
  • Managing travel and expenses of employees on their business trips

The database design includes the following features:

  • Employee travel request system with booking preferences
  • Approval workflow based on business travel purpose and preferences
  • Company policy integration (airfare limits, flight class, hotel ratings, etc.)
  • Comprehensive booking information tracking
  • Travel expense reporting and tracking
  • Reimbursement process aligned with company travel policy and approval workflow

Technologies: MySQL, Lucid Chart, MySQL Workbench
